Analysis of Titanic Data¶

Questions:¶

  1. Adult females and children had preference in the lifeboats. Statistically speaking, was there more chance of survival if you were an adult female or a child rather than an adult male?
  2. Was there a dependency in the survival rate given the social class in which each age category falls on, while age categories are Adult Females, Adult Males, and Children; and the social classes are 1, upper; 2, medium and; 3, lower.

Description of Investigative Process:¶

  1. I reviewed the data and the dictionary about its metadata to understand the sample more.

  2. I brainstormed and formulated the first iteration of questions mentioned above. Initially, it was only one.

  3. I set my scope by thinking on possible outcomes for this analysis.

  4. Then I chose the method that I thought it was most applicable for this dataset: Naive Bayes Theorem.

  5. I first categorized the data by using just the Sex. However, I thought it was going to be more meaningful if I change it to Age Category by adding children who are younger than 18. Thus, I modified the first question mentioned above.

  6. I started validating the data by assigning Nones to any null value in affected columns. I converted any other data to the most appropriate data type.

  7. Then, I decided which records to delete depending on how those data errors were going to affect my analysis.

  8. I created the final table ready for analysis and investigate the results for final validation.

  9. I started brainstorming for the best ways to visualize my data.

  10. I iterated through different scenarios of visualization until I found the best-fitted ones.

  11. I analyzed the visualization and came up with another question to ask. Thus, I added another question to the questions above.

  12. I brainstormed the best ways to visualize the new question and how to improve the function to create the data frame to be fed to the visualizations.

  13. I verified the data by manually checking probabilities and proportions manually on a white board by taking a sample of the sample.

  14. I formulated the conclusions.

# Interates through values and if there is a blank instance returns none to avoid an error

def par_int_float_none(value, **parse_type):
    
    #When the user leaves the parse_type optional argument, the function will ignore it
    if parse_type['parse_type'] != {}:
        parse_type = parse_type['parse_type']['convertype']
    
    #Converts the values provided according to the parse_type
    #Return None for missing blank values
    #Return the same value type for opted-out parse_type
    if value == ' ' or value == '' or value == None:
        return None    
    elif parse_type == 'float':
        return float(value)
    elif parse_type == 'int':
        return int(value)
    else:
        return value

#Creates a table of details statistics
def stat_detail_frame(data, category_array, what_toassess):
    
    #converting to series
    total = data.groupby(category_array)[what_toassess].count()
    partial = data.groupby(category_array)[what_toassess].sum()
    
    #Getting data into dataframe
    dataframe_conv = pd.DataFrame(total)

    #Creates and renames columns
    dataframe_conv.columns = ['Total Embarked']
    dataframe_conv.loc(axis = 1)['Total Survived'] = partial

    dataframe_conv.loc(axis = 1)['Total Died'] = dataframe_conv.iloc(axis = 0)[:, 0] - dataframe_conv.iloc(axis = 0)[:, 1]

    dataframe_conv.loc(axis = 1)['Survival Ratio'] = round((dataframe_conv.iloc(axis = 0)[:, 1] / dataframe_conv.iloc(axis = 0)\
                                                            [:, 0]), 2)

    dataframe_conv.loc(axis = 1)['Proportion of Ratio'] = round(dataframe_conv.iloc(axis = 0)[:, 3] / dataframe_conv.iloc(axis = 0)\
                                                 [:, 3].sum(), 4) * 100

    return dataframe_conv

Validating that 177 values are missing

Data Wranglign Notes: We found that 177 records in the sample had no Age data. This could have been because the passenger refused to provide such data, which could indicate (by suggestion and assuming that older people would refuse to provide their age to a stranger) that the age was too high if these values were not missing at random.

A couple of ways to address this issue is by assigning the mean or the median value to the missing ages. However, given that our sample size is 819 and our analysis will be multivariate, we should just follow the omission technique instead of the data imputation. The total sample left should be 714 at random, which still is a representative sample.

An argument about this decision is that we can greatly create a bias by adding centralized values when the actual values could have been skewed and change the desired outcome, which is trying to predict the true proportions of survival or death depending on the Age Category, in the first scenario; and trying to predict the true proportions of survival or death depending on the Age Category and Social Class, in the second scenario.

Validation Assessment: In the data-frame above, records without age data would affect the rest of the analysis. Since Adult Females and Children were given preference on board the lifeboats, the hypothesis that Children and Adult Females had more chance of survival is applicable. This claim should be assessed to strengthen this claim.

Action: Since there is no strong evidence that these null values under the age metadata were random or simply these people decided not to provide such information, these records will be omitted from the analysis to avoid creating a bias and changing our desired outcome.

Conclusion¶

In conclusion, based on the Naive Bayes Theorem, there is more likelihood that an Adult Female will have more chances of survival than Children and Adult Males. However, although the likelihood of being an Adult Female who survived is greater than that of being a Child who survived, the differences between both may not have enough significant difference. A further hypothesis test of their proportions is recommended.

Nevertheless, to answer our first question, there is strong visual evidence that Adult Females and Children had more chance of survival than men.

Also, when including social classes as the subcategory, within each age category there is enough statistical and visual evidence that being in first class was a factor in having more chances of survival on each age category. However, when it came to second social class, Adult Females and Children could not show too much significance difference, specifically in the Children age category. There was clearly enough significant difference between the probability surviving given that an Adult Male was in first social class than being in second social class. Therefore, it seems that the social class of people who embarked was a factor of survival in the Titanic. A further hypothesis test of homogeneity is recommended.
